TwitterにアクセスしてハッシュタグのTop10を表示する Scalaプログラム 
-------------------------------------------------------------------


1) simpleapp1.zip を適当な場所に解凍し、そのディレクトリに cd 

2) src/main/scala/SimpleApp.scala 内の以下の文字列を適切な文字列に書き換える
  
YourTwitterConsumerKey         
YourTwitterConsumerSecret      
YourTwitterAccessToken
YourTwitterAccessTokenSecret   
                                                                                                                                                                                  
3) sbt package にてコンパイル、jarファイルの作成
  
4) sbt run にて実行
